# Figure Selection Rules

Select visuals for presentation use, not for archival completeness.

## Prefer figures that:
- support a clear and specific message
- show comparison, separation, trend, or anomaly clearly
- can be explained in one slide with minimal extra context
- are readable without zooming excessively
- are not redundant with other selected figures

## Move figures to backup when:
- they support the same message as a clearer figure
- they require too much technical explanation
- the visual is useful but not essential to the main storyline
- the meaning is likely valid but still somewhat uncertain

## Omit figures when:
- they repeat the same point with no additional value
- the visual is too noisy or illegible
- the message cannot be stated clearly
- the relationship to the main reporting goal is weak

## Default priority labels
- high: essential for the main story
- medium: useful but not critical
- low: optional or backup-oriented

## General rule
Prefer one strong figure over three similar figures.
