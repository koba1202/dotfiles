# 常に日本語で対応

- 全ての説明と回答は日本語で提供する
- 思考は英語に翻訳して行うこと
- コードコメントも日本語で記述する
- エラーメッセージの説明も日本語で行う

# ユーザーの判断を求める場合はAskUserQuestionを利用

ユーザーの判断を求める場合はAskUserQuestionツールを使うこと。

# 絵文字禁止

ファイル出力する場合絵文字は利用しない。
また、会話セッションにおける絵文字利用は不要。

## Python スクリプトを実装する時の意識

- 基本的にimportなどはプロジェクトルートから索引するようにすること
- 画像やテキストなどアウトプットすることがあればプロジェクトルートから指定すること
- プロジェクトルートがわからない場合はユーザーに確認をとること

## ファイル削除について

ファイル削除は `settings.json` の権限でも制限しているが、Claudeはファイルを削除できないようにしている。
そのためファイル削除のタスクがあればユーザーに委譲するようにすること。
